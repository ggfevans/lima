# Changelog

## v0.1.0

Initial release.

- Real-time LinkedIn messaging via SSE
- Conversation list with All/Unread filtering
- Thread view with grouped sender headers
- Compose and reply
- Mark read/unread, delete conversation
- Keyboard-driven navigation (vim-style + arrows)
- LinkedIn session cookie auth with secure credential storage
- Dracula (dark) and LinkedIn (light) themes
- Responsive layout (two-panel / single-panel)
